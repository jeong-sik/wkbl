## 🚀 Quick Wins (즉시 적용 가능)

1. **슛 차트 SVG** - 코트 이미지 + 클릭 좌표
2. **PWA manifest** - 이미 HTMX 사용 중, 쉬움
3. **공유 카드** - 서버사이드 이미지 생성
4. **레이더 차트** - Chart.js 또는 순수 SVG
5. **다크모드 개선** - CSS 변수만 조정
